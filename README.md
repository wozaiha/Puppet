# Puppeteer Plus Pro Ultra Max

## 主要功能

* 基础的Pupperteer功能
* 白名单
* 简便的Gla预设导出及分享、查看
* 个人添加的转义功能
  * 支持纯字符转义（包括正则）
  * 支持Gla预设/单件、C+预设、Moodles效果及预设（请务必仔细查看提示文字）
